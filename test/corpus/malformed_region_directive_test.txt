================================================================================
Malformed region directive with space
================================================================================

codeunit 50100 "Test"
{
    # Region TODO: This is malformed
    procedure Test()
    begin
    end;
    #endregion
}

--------------------------------------------------------------------------------

Note: The directive "# Region" (with space) is malformed. The correct syntax 
is "#region" (no space). This will cause a parsing error. The parser correctly
identifies this as an error since it's not valid AL syntax.

================================================================================
Correct region directive syntax
================================================================================

codeunit 50100 "Test"
{
    #region TODO: This is correct
    procedure Test()
    begin
    end;
    #endregion
}

--------------------------------------------------------------------------------
